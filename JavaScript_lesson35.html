1. Đối tượng (Object) là gì?

Trong Javascript, đối tượng là một khái niệm trừu tượng thể hiện cho một đối tượng cụ thể và nó có sẵn một số đối tượng như Date, Number. Ngoài các đối tượng này thì lập trình viên có thể tự tạo một hoặc nhiều đối tượng theo ý của mình dựa trên yêu cầu của ứng dụng. Ví dụ cần tạo một đối tượng chuyên xử lý vấn đề về bình luận cho trang tin tức thì mình sẽ tạo một đối tượng Comment.

Khởi tạo đối tượng

Để tạo một đối tượng trong Javascript, ta có hai cách sau.

Cách 1: Sử dụng từ khoá new Object():

var Comment = new Object();

Cách 2: Sử dụng từ khóa {}:

var Comment = {};

2. Thuộc tính và phương thức đối tượng

Mỗi đối tượng sẽ có các thuộc tính và phương thức.

Thuộc tính

Thuộc tính là những đặc điểm (có thể là biến) cần lưu trữ trong đối tượng. Ví dụ với đối tượng Comment có thể cần các thuộc tính sau:

  • title
  • content
  • fullname
  • email

Để khai báo thuộc tính, ta có 3 cách.

Cách 1: Sử dụng từ khóa new Object():

// Khởi tạo
var Comment = new Object();

// Thêm thuộc tính
Comment.title = '';
Comment.content = '';
Comment.fullname = '';
Comment.email = '';

Cách 2: Sử dụng từ khóa {} và thêm thuộc tính ngay lúc khai báo:

// Khởi tạo
var Comment = {
// Thêm thuộc tính
title : "",
content : "",
fullname : "",
email : ""
};

Cách 3: Sử dụng từ khóa {} và thêm thuộc tính sau đó:

// Khởi tạo
var Comment = {};

// Thêm thuộc tính
Comment.title = '';
Comment.content = '';
Comment.fullname = '';
Comment.email = '';

Phương thức

Phương thức là những hành động (có thể hiểu là hàm) của đối tượng. Ví dụ trong đối tượng Comment thì ta cần hai phương thức sau:

  • addComment()
  • editComment()

Lúc này ta sẽ có ba cách khai báo tương tự như cách khai báo thuộc tính.

Cách 1: Sử dụng từ khóa new Object():

// Khởi tạo
var Comment = new Object();

// Thêm phương thức
Comment.addComment = function() {
// Do something
};
Comment.editComment = function() {
// Do something
};

Cách 2: Sử dụng từ khóa {} và thêm thuộc tính ngay lúc khai báo:

// Khởi tạo
var Comment = {
// Thêm phương thức
Comment.addComment = function() {
// Do something
};
Comment.editComment = function() {
// Do something
};
};

Cách 3: Sử dụng từ khóa {} và thêm thuộc tính sau đó:

// Khởi tạo
var Comment = new Object();

// Thêm phương thức
Comment.addComment = function() {
// Do something
};

Comment.addComment = function() {
// Do something
};

Thao tác với thuộc tính và phương thức đối tượng

Sau khi tạo xong đối tượng thì ta có hai cách sử dụng căn bản đó là gọi và gán dữ liệu cho thuộc tính và phương thức. Nhưng nếu xem xét mọi khía cạnh thì chúng ta có các thao tác thông thường sau:

  • Gán giá trị thuộc tính
  • Lấy giá trị thuộc tính
  • Gọi phương thức

Gán giá trị cho thuộc tính

Để gán giá trị cho thuộc tính chúng ta chỉ việc thực hiện bằng cách sử dụng toán tử = giống như cách gán giá trị cho biến.

Comment.title = "Tiêu đề bình luận";

Nhưng nếu bạn gọi từ một hàm trong đối tượng thì bạn có thể sử dụng từ khóa this:

var Comment = {
title : "",
addComment : function() {
// Sử dụng this chỉ đối tượng
this.title = "Đây là tiêu đề";
}
};

Lấy giá trị thuộc tính

Để lấy giá trị thuộc tính thì ta làm tương tự như thao tác với biến.

var title1 = Comment.title;

Tương tự, nếu gọi từ một hàm trong đối tượng thì ta có thể sử dụng từ khóa this:

var Comment = {
title : "",
addComment : function() {
// Sử dụng this chỉ đối tượng
var title1 = this.title;
}
};

Gọi phương thức

Tương tự như thuộc tính:

Comment.addComment();

Gọi trong hàm của đối tượng:

var Comment = {
title : "",
addComment : function() {
this.editComment();
},
editComment : function() {
this.addComment();
}
 };

4. Mảng chứa đối tượng - đối tượng chứa đối tượng

Đối tượng chứa đối tượng

Ví dụ 1: Gom các thuộc tính của Comment vào một đối tượng Info.

var Comment = {
Info : {
title : "",
content : "",
fullname : "",
email : ""
}
};

Ví dụ 2: Gom các thuộc tính của Comment vào một đối tượng Func.

var Comment = {
Func : {
addComment : function() {
// Do something
},
editComment : function() {
// Do something
}
}
}

Lúc này để truy xuất tới các thuộc tính và phương thức này, ta chỉ việc sử dụng dấu chấm . và bổ sung thêm một cấp nữa:

Comment.Info.fullname = "Đào Lập Xuân";
Comment.Func.editComment();

Mảng chứa đối tượng

Để gán giá trị là một đối tượng vào mảng cũng tương tự như gán các giá trị bình thường khác (xem mảng trong JavaScript).

// Đối tượng Comment
var Comment = {
title : "",
content : "",
fullname : "",
email : ""
};

// Khởi tạo mảng
var arrayComments = [];

// Gán giá trị cho phần tử mảng
arrayComments[0] = Comment;

// Gọi tới thuộc tính
arrayComments[0].title = "Tiêu đề bình luận";
// Kiểm tra thử
alert(arrayComments[0].title);

Hoặc ta có thể chèn cập dấu [ ] vào khai báo của đối tượng, ý nói rằng đây là một mảng các đối tượng:

// Mảng các đối tượng Comment
var Comment = [{
title : "",
content : "",
fullname : "",
email : ""
}];

// Gọi tới thuộc tính
arrayComments[0].title = "Tiêu đề bình luận";
// Kiểm tra thử
alert(arrayComments[0].title);
