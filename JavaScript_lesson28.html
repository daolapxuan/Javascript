1. Chuỗi (String)

Chuỗi là một đoạn text có thể có một hoặc nhiều ký tự và thông thường chúng ta sẽ lưu trữ nó vào một biến, biến này ta sẽ gọi là biên có kiểu dữ liệu String (chuỗi). Tất cả các chuỗi đều phải được bao quanh bởi dấu nháy đơn '' hoặc dấu nháy kép "".

Ví dụ:

var name = "Đào Lập Xuân";
var email = 'daolapxuan@gmail.com';

Trường hợp trong chuỗi cũng có xuất hiện dấu nháy đơn hoặc dấu nháy đôi thì bắt buộc phải thêm ký tự \ đằng trước dấu nháy đó nếu ko sẽ bị lỗi cú pháp.

Ví dụ:

var name = "Đào \"Lập Xuân";
var email = 'daolapxuan@\'gmail.com';

Ngoài ra còn có các kí hiệu kết hợp với dấu \:

Code

2. Nối chuỗi

Để nối chuỗi ta sử dụng dấu + để ghép 2 chuỗi (hoặc biến) với nhau.

Ví dụ:

var message = "Toi ten la" + " Dao Lap Xuan!";

Hoặc:

var message1 = "Toi ten la";
var message2 = "Dao Lap Xuan!";

var message = message1 + message2;

Note: khi xuống hàng

Ta phải sử dụng dấu + để nối chuỗi khi ta enter, nếu ko bị lỗi cú pháp.

Ví dụ:

// Đúng
var message1 = "My name is"
+ "Dao Lap Xuan";
// Sai
var message2 = "My birthday is"
February 4th, 1993";

Hoặc nếu muốn viết gọn hơn thì sử dụng dấu \ để báo cho trình duyệt biết là có xuống hàng.

Ví dụ:

var message = "I'm from VietNam \
I'm Vietnamese!";

Nếu muốn ép một giá trị nào đó sang kiểu string, thì có thể sử dụng cú pháp: string.toString().

Ngoài cách trên bạn có thể sử dụng đối tượng String để tạo mới chuỗi và vì là đối tượng nên nó phải có từ khóa new đặt đằng trước.
