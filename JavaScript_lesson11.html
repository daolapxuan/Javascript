1. DOM là gì?

DOM là viết tắt của chữ Document Object Model, có thể tạm hiểu là mô hình các đối tượng trong tài liệu HTML.

Trong mỗi thẻ HTML sẽ có những thuộc tính (Properties) và có phần cấp "Cha - Con" với các thẻ HTML khác. Sự phân cấp và các thuộc tính của thẻ HTML này goi là selector và trong DOM có nhiệm vụ xử lý các vấn đề như đổi thuộc tính của thẻ, đổi cấu trúc HTML của thẻ,...

Trong Javascript, để thao tác với các thẻ HTML, ta thông qua đối tượng document.

2. Các thể loại DOM trong Javascript

Danh sách chia nhóm DOM:

  • DOM document: có nv lưu trữ toàn bộ các thành phần trong tài liệu của website.
  • DOM element: có nv truy xuất tới các thẻ HTML nào đó thông qua các thuộc tính như id, class, tên thẻ HTML, selector,...
  • DOM HTML: có nv thay đổi giá trị nội dung và giá trị thuộc tính của các thẻ HTML.
  • DOM CSS: có nv thay đổi các định dạng CSS của thẻ HTML.
  • DOM Event: có nv gán các sự kiện như onclick(), onload() vào các thẻ HTML .
  • DOM Listener: có nv lắng nghe các sự kiện tác động lên thẻ HTML đó.
  • DOM Navigation dùng để quản lý, thao tác với các thẻ HTML, thể hiện mối qua hệ CHA - CON của các thẻ HTML.
  • DOM Node, Nodelist: có nhiệm vụ thao tác với HTML thông qua đối tượng (Object)

3. Tìm hiểu DOM element

Tìm thẻ HTML theo ID

Cú pháp:
var element = document.getElementById('id');

Lấy id thẻ input:
var element = document.getElementById('id-input');

Lấy giá trị thẻ input:
document.write(element.value);

Tìm thẻ HTML theo tên thẻ HTML

Tên thẻ HTML ở đây là các thẻ như p, h1..h6, div,...

Cú pháp:
var element = document.getElementById('tagName');

Lấy tagName thẻ input:
var element = document.getElementsByTagName('input');

Lấy giá trị thẻ input:
document.write(element[0].value);
document.write(element[1].value);

Note: Vì lấy theo tagName HTML thì sẽ bị trùng nhiều phần tử do một thẻ html đc sử dụng nhiều lần, nên giá trị trả về sẽ là một mảng object, ta phải đánh số thứ tự như mảng mới lấy đc value cần lấy.
Ví dụ ở đây, element[0] sẽ trả về giá trị thẻ input 1 có id là id-input, và element[1] sẽ trả về value của thẻ input 2 có id là id-input1.

Tìm thẻ HTML theo class

Cú pháp:
var element = document.getElementsByClass('class');

Lấy class thẻ input:
var element = document.getElementsByClass('class-input');

Lấy giá trị thẻ input:
document.write(element[0].value);
document.write(element[1].value);

Note: Tương tự lấy theo tagName HTML, lấy theo class cũng trả về mảng các object.
Ví dụ ở đây là, trả về 2 value của 2 thẻ input có cùng class là class-input
