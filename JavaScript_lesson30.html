Đối tượng Number

Trong Javascript có hai giá trị lưu trữ trong Number đó là có dấu chấm động và không có dấu chấm động.

Ví dụ:

var x = 12; // Ko có dấu chấm động
var x = 11.5; // Có dấu chấm động

Nếu một số quá lớn thì ta có thể dùng số mũ để biểu diễn:

var x = 123e5 // Là 123 x 10^5
var y = 123e-5 // Là 123 x 10^-5

Các số trong Js luôn luôn là 64 bít và kiểu float, vì vậy bạn không thể định nghĩa các kiểu dữ liệu như integer, shor, long, double,... hay nói cách khác, trong Js khi làm việc với chữ số thì chỉ có khái niệm Number.

Ngoài ra ta có thể biểu diện giá trị Number ở dạng nhị phân, thập phân, thập lục phân,...

Ví dụ:

var x = 0xFF; // x có giá trị 255

Vì các dữ liệu liên qua đến chữ số đều có kiểu dữ liệu Number, nên khi dùng typeof sẽ trả về là 'number'.

Chuyển Number sang String

Để chuyển một biến đang ở dạng Number sang kiểu String thì ta sử dụng phương thức number.toString(type), hàm này có một tham số truyền vào là type và đây chính là kiểu dữ liệu muốn chuyển về, mặc định là hệ thập phân (10).

Ví dụ: Nhập một số từ bàn phím, kiểm tra số đó là kiểu gì bằng hàm typeof. Tiếp theo đổi số đó sang giá trị nhị phân, bát phân, thập phân và thập lục phân.

var myNumber = 128;
var typeOf = typeof myNumber; myNumber.toString(); // return 128
myNumber.toString(16); // return 80
myNumber.toString(8); // return 200
myNumber.toString(2); // return 10000000

Số Infinity

Infinity cũng là một kiểu dữ liệu Number và khi một biến có giá trị là Infinity thì tức là nó đã vượt mức lưu trữ cho phép nên theo mặc định nó sẽ chuyển về dạng đó. Vì nó cũng là một giá trị nên bạn có thể so sánh bình thường.

NaN - Not a Number

Nếu ta thực hiện một phép toán nào đó có liên quan kiểu Number, nhưng vi phạm quy tắc tính toán thì kết quả sẽ trả về một giá trị gọi là NaN (Not a Number).

Ví dụ:

var x = 2 / "ABC"; // NaN

Tuy nhiên nếu mẫu số là một "chuỗi" Number thì kết quả nó vẫn hoạt động bình thường

Ví dụ:

var x = 2 / "2"; // = 1

Để kiểm tra một biến nào đó có phải là NaN hay không, ta sử dụng hàm isNaN().

Ví dụ:

var x = 2 / "a";
if (isNaN(x)) {
document.write("NaN");
}
