Các hàm xử lý có thể chia làm hai nhóm:

Nhóm toàn cục: là những hàm ko nằm trong đối tượng Number

  • Number(type) trả về một number và định dạng cơ số type=(nhị phân, bát phân, thập phân, thập lục phân).
  • parseFloat() chuyển sang một số Float.
  • parseInt() chuyển sang một số Integer.

Nhóm toàn cục: là những hàm nằm trong đối tượng Number.

  • toString() chuyển sang một số Integer.
  • toFixed(n) chuyển sang số có n số lẻ ở bằng nhau.
  • toPrecision(n) chuyển sang số có chiểu dài là n.
  • valueOf() là giá trị của biến hoặc một giá trị nào đó.

1. Hàm xử lý Number toàn cục

Number

Dùng để chuyển đổi một biến hoặc một giá trị nào đó sang kiểu number, nó chuyển tất cả các định dạng như Boolean, Date, String. Nếu trường hợp giá trị cần chuyển đổi ko thể chuyển sang Number được thì nó chuyển sang giá trị mặc định là NaN.

Ví dụ:

var booleanTrue = true;
Number(booleanTrue); // return 1

var booleanFalse = false;
Number(booleanFalse); // return 0

var stringString = 'Dao Lap Xuan';
Number(stringString); // return NaN

var stringNumber = "20";
Number(stringNumber); // return 20

var date = new Date();
Number(date); // return 1

parseInt()

Hàm này chuyển dữ liệu sang định dạng Integer, có tác dụng như hàm Number(), nhưng có vài điểm khác biệt sau:

  • Nếu chuỗi có các ký tự đầu tiên là các con số và đằng sau là chữ cái thì nó sẽ lấy các số đầu tiên đó và chuyển thành kiểu number. Nhưng nếu dùng hàm Number() nó sẽ trả về NaN.
  • Nếu dữ liệu ở các định dạng khác string thì nó chuyển thành NaN.

Ví dụ:

var booleanTrue = true;
Number(booleanTrue); // return NaN

var booleanFalse = false;
Number(booleanFalse); // return NaN

var stringString = '1234 Dao Lap Xuan';
Number(stringString); // return 1234

var stringNumber = "20";
Number(stringNumber); // return 20

var date = new Date();
Number(date); // return NaN

parseFloat()

Hàm này chuyển dữ liệu sang định dạng float, về cách sử dụng y hệt hàm parseInt().

Ví dụ:

var booleanTrue = true;
Number(booleanTrue); // return NaN

var booleanFalse = false;
Number(booleanFalse); // return NaN

var stringString = '123.4 Dao Lap Xuan';
Number(stringString); // return 123.4

var stringNumber = "-20.1";
Number(stringNumber); // return -20.1

var date = new Date();
Number(date); // return NaN

2. Hàm xử lý Number cục bộ

Những hàm cục bộ phải gắn liền với đối tượng Number cụ thể, ví dụ bạn khởi tạo một biến var x = 10; thì lúc này x có tất cả các hàm cuc65 bộ. Ngoài ra ta có thể sử dụng cặp mở đóng () để bao quanh một biểu thức hoặc một giá trị thì vẫn sử dụng bình thường.

Ví dụ:

var x = 123; // typeof x = number
x.toString(); // typeof x = string
(456).toString();
(1 + 2 * 3 / 4).toString();

toString()

Hàm toString() có tác dụng chuyển đổi Number sang kiểu String.

Ví dụ:

var x = 123;
typeof x = number // number

x.toString();
typeof x = string // string

typeof 456; // number
typeof (456).toString() // string

typeof (1 + 2 * 3 / 4).toString(); // string

toFixed(n)

Hàm này có tác dụng chuyển một số sang một số có n số lẻ ở sau nó và có làm tròn.

Ví dụ:

var x = 123.456;

x.toFixed(0); // return 123
x.toFixed(1); // return 123.5
x.toFixed(2); // return 123.46
x.toFixed(3); // return 123.456
x.toFixed(6); // return 123.456000

toPrecision(n)

Hàm này cũng gần giống hàm Fixed(n), nhưng khác ở chỗ nó chuyển 1 số thành số có chiều dài n (chiều dài gồm phần nguyên và phần thực).

var x = 123.456;

x.toPrecision(); // return 123.456
x.toPrecision(2); // return 12
x.toPrecision(4); // return 123.5
x.toPrecision(5); // return 123.46
x.toPrecision(6); // return 123.456
x.toPrecision(9); // return 123.456000

Hàm valueOf()

Hàm valueOf() có tác dụng lấy giá trị của một biến hoặc một giá trị khác - ý nghĩa là lấy giá trị của (cái gì đó).

Hàm này thực chất ko đc sử dụng nhiều, vì thông thường ta lấy giá trị trực tiếp luôn.

Ví dụ:

var x = 123 + 123;

x.valueOf(); // return 246

(1 + 2 * 3).valueOf; // return 7
