Đối tượng Date

Tất cả các vấn đề liên quan đến thời gian đều nằm trong đối tượng Date, vì vậy để khởi tạo đối tượng Date, thì bạn sử dụng cú pháp:

var timeObj = new Date();

Đây là cách thông thường, khi dùng cách này nó sẽ lấy thời gian hiện tại trên máy của bạn (client), điều này có nghĩa mỗi Client sẽ có thời gian phụ thuộc vào máy tính của họ.

Ví dụ: Lấy thời gian hiện tại của máy tính.

Thời gian máy tính:

Dưới đây là bốn cách khởi tạo thông thường nhất:

// Thời gian hiện tại
new Date();

// Tham số truyền vào mili giây
new Date(miliseconds);

// Tham số truyền vào là chuỗi string ngày tháng

new Date(dateString);

// Tham số truyền vào bao gồm:
// - year
// - month
// - day
// - hours
// - minutes
// - seconds
// - miliseconds
new Date(year, month, day, hours, minutes, seconds, miliseconds);

Khi bạn truyền vào tham số thì đối tượng đó sẽ tự nhận diện và chuyển đổi về đúng định dạng ngày tháng nên bạn hoàn toàn yên tâm sử dụng.

Ví dụ:

var dateObj = new Date(1993, 2, 4); // February 4th, 1993

Định dạng (format)

Có 3 định dạng chính:

  • ISO
  • Long
  • Short

ISO

Định dạng chuẩn của ISO 8601 là (YYYY-MM-DD) hoặc (YYYY-MM) hoặc (YYYY).

Nếu truyền ko đủ (ngày - tháng - năm - giờ - phút - giây) thì mặc định các tham số khác sẽ lấy minimum time.

Ví dụ:

var dateISO_1 = new Date("1993-2-4"); // Thu Feb 04 1993 00:00:00 GMT+0700 (SE Asia Standard Time)
var dateISO_2 = new Date("1993-2"); // Mon Feb 01 1993 00:00:00 GMT+0700 (SE Asia Standard Time)
var dateISO_3 = new Date("1993"); // Fri Jan 01 1993 07:00:00 GMT+0700 (SE Asia Standard Time)

Long

Định dạng này nghĩa là bạn truyền vào với tên tháng là ba chữ cái đầu tiên ghi bằng tiếng Anh. Lúc này có thể tùy ý đặt ví trí vì đối tượng Date tự nhận diện.

Ví dụ:

var dateLong_1 = new Date("February 1993 4");
var dateLong_2 = new Date("4 February 1993");
var dateLong_3 = new Date("1993 4 February");

Khi chạy code trên thì cho ra kết quả như nhau.

Short

Định dạng này đươc lưu trữ dưới dạng MM/DD/YYYY hoặc YYYY/MM/DD hoặc MM-DD-YYYY hoặc YYYY-MM-DD.

var dateShort_1 = new Date("02/04/1993");
var dateShort_2 = new Date("1993/02/04");
var dateShort_3 = new Date("02-04-1993");
var dateShort_4 = new Date("1993-02-04");

Định dạng đầy đủ

Trên là những định dạng ghi tắt, nếu truyền đầy đủ thì phải gõ (ngày - tháng - năm - giờ - phút - giây - timezone):

var date = new Date("Thu Feb 04 1993 19:07:49 GMT+0700 (SE Asia Standard Time)");
