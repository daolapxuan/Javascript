Thay đổi CSS bằng Javascript

Cú pháp set CSS bằng Javascript:

document.getElementById('object').style.cssName = 'something';

Cú pháp get CSS bằng Javascript:

var value = document.getElementById('object').style.cssName;

Note: trường hợp thuộc tính css name có dấu gạch ngang như: font-family, line-height, margin-bottom,... thì thuộc tính đó trong style (javascript) có tên là fontFamily, lineHeight, marginBottom,...
Nghĩa là sẽ bỏ dấu gạch ngang và viết hoa ký tự đầu tiên sau dấu gạch ngang.

Ví dụ:

document.getElementById('object').style.fontFamily = ('something');

document.getElementById('object').style.lineHeight = ('something');

document.getElementById('object').style.marginBottom = ('something');

Ví dụ thay đổi CSS bằng Javascript

Ví dụ 1.

Viết một chương trình gồm 4 buttons và 1 thẻ div, khi click vào từng button thì sẽ thiết lập màu sắc, background, chiều cao, font-size của thẻ div.

Đây là nội dung cần thay đổi bằng CSS

Ví dụ 2.

Viết chương trình đăng nhập và validate thông báo username, password. Nếu user không nhập username hoặc mật khẩu thì hiển thị thông báo message trong một thẻ div và chữ màu đỏ, ngược lại thì thông báo validate thành công và chữ màu xanh.
