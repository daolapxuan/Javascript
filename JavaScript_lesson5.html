Lệnh kiểm tra điều kiện

Nhập a, b từ bàn phím và nhấn xem kết quả

Nhập a =
Nhập b =

Lệnh If đơn giản

Kết quả sẽ hiện ra nếu a = b, ngược lại thì ko có gì cả.

Lệnh If else đơn giản

Kết quả sẽ in ra a = b, nếu a = b và ngược lại sẽ in ra a != b

Nhiều lệnh If else

Kết quả sẽ in ra nhiều trường hợp khác nhau

Lệnh If else lồng nhau

Ta so sánh a với 10, nếu a > 10 thì sẽ so sánh tiếp a với b và in ra các kết quả tương ứng
