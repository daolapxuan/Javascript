Vòng lặp While và Do-while

Nhập n từ bàn phím

n =

Vòng lặp while từ 1 đến n:

var i = 1;
while (i <= n) {
document.write(i + ' ');
i++;
}

Vòng lặp while từ n giảm tới 1:

var i = n;
while (i >= 1) {
document.write(i + ' ');
i--;
}
